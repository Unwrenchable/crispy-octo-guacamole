# AgentX Pack for crispy-octo-guacamole

This repository is upgraded with AgentX capabilities.

## Included
- `agents.json`: merged core + imported + custom per-repo agents
- `access_profiles.json`: safe/balanced/power profiles
- `agency_import.json`: imported agents from agency-agents

## Suggested commands
```bash
agentx find crispy-octo-guacamole
agentx check crispy-octo-guacamole-implementation-pilot --profile balanced
agentx check crispy-octo-guacamole-orchestrator --profile power
```
